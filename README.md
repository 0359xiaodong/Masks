Masks
====

A collection of TextWatcher for Android, it comes with CPF/CNPJ formatting, and a Brazil telephone, that's all!

Gradle:
===
```groovy
dependencies {
   compile 'com.github.leonardoxh:android-masks:1.1'
}
```
